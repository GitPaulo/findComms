findComms

findComms is a tool that searches twitter profiles for artists with stores/commissions/etc.

findComms is powered by the public twitter API and therefore is limited by its rates.

At the moment, users with followers in the thousands will not have full searches. (WIP)

Usage

Simply search a user by id or display name (https://twitter.com/this_part) You can filter by adding the selector ':', cases:

  • :following
  • :followers
    • *the default search space is both.

Example: MyDisplayName:followers

That will search all of "MyDisplayName's" followers for artists.
